# One ROM Fire 28 pin

**Verified - This design only differs from A2 in correcting the SWD CLK/DIO silkscreen labels.

Note that the RP2354 variant is **unverified**.

A3 revision of the 28 pin Fire ROM board.  Combined USB/Pro version with USB-C and SWD header pins.

Very minor difference from revision A2:

- CLK/DIO silkscreen labels swapped to correct positions.

There are two versions of the BOM/POS files:

- The original version with an RP2350A and external flash - **verified**
- A lower cost version with an RP2354A and no external flash - **unverified**
