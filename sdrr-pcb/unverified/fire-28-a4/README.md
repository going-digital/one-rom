# One ROM Fire 28 pin

**Unverified**

A4 revision of the 28 pin Fire ROM board.  Combined USB/Pro version with USB-C and SWD header pins.

This design only differs from A3 in a minor silkscreen clarification (see #121).

There are two versions of the BOM/POS files, which are unchanged from A3.

- The original version with an RP2350A and external flash
- A lower cost version with an RP2354A and no external flash
